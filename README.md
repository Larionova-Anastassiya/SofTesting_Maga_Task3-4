# SofTesting_Maga_Task3-4


Test Cases Description (DemoBlaze)

1. Sign Up Test:
   - Objective: To verify successful registration of a new user.
   - Steps: Navigate to the homepage, click "Sign up", enter username/password, click "Sign up" button, handle the alert (success/user already exists).

2. Login Test:
   - Objective: To verify successful login of an existing user.
   - Steps: Navigate to the homepage, click "Log in", enter username/password, click the "Log in" button.
   - Expected Result: The text "Welcome [Username]" appears on the page.

3. Logout Test:
   - Objective: To verify successful user logout.
   - Steps: Click "Log out".
   - Expected Result: The "Log in" button becomes visible again.

4. Login Again Test:
   - Objective: Re-verification of the user login functionality.
   - Steps: See Login Test.

5. Buy Product Test:
   - Objective: To verify the ability to add a product to the cart and see it there.
   - Steps: Select the product "Samsung galaxy s6", click "Add to cart", confirm the alert, navigate to "Cart".
   - Expected Result: "Samsung galaxy s6" is displayed in the cart.
